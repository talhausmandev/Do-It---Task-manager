## 🌟 Do It -- Task Manager


**Do-it** is a **free and simple** web app where users can add their **todos** and save these in **localstorage** also it offers to add **priority** and **due date**

🚀 Built with **HTML**, **CSS**, and **Javascript**.

---

## ✨ Features

- 🆓 **Free to use**
- 🔗 **Add your todos**
- 📝 **Add priority of todo**
- 🖼️ **Add due date to complete task**
- ⚡ **clean, and easy to use**
- 😍 **Beautiful Design**
